# short-code - Sample Express

### What's this
[Express](http://expressjs.com/)のGettingStartedをお試し。  
qiitaの記事を適当に取得して表示させるところまで。

### How to start

1. git clone する
2. `npm install`
3. `npm run express`
4. `http://localhost:3000/`にアクセスする


### Notes
ExpressのソースがES6に対応していないので、ES5の記述をしている(ビルドスクリプトは使わないし動かない)。

### Libraries

* Express
* ejs
* nodemon
* superagent
